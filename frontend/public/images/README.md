# Wedding Images

This directory contains images for the wedding website.

## Required Images

- `hero-wedding.jpg` - Main hero background image for the homepage
  - Recommended size: 1920x1080 or larger
  - Format: JPG or PNG
  - Should be a beautiful wedding-related image (venue, couple, etc.)

## Usage

Images in this directory are served from `/images/` path in the application.

Example:

- File: `public/images/hero-wedding.jpg`
- URL: `/images/hero-wedding.jpg`
